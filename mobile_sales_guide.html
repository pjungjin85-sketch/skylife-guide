🗣️ 고객 접근 & 첫 대화 흐름

1
자연스러운 인사 & 소속 안내
"안녕하세요, KT 스카이라이프 OOO대리점 소속 OOO입니다." — 소속을 명확히 밝혀 신뢰감 형성
2
현재 이용 통신사 & 요금 파악
현재 쓰시는 통신사, 월 요금, 데이터 사용량을 자연스럽게 확인합니다.
💡 "혹시 한 달에 데이터 얼마나 쓰세요?" — 부담 없는 질문으로 시작
3
불만족 포인트 파악
요금이 비싸다 / 속도가 느리다 / 혜택이 없다 등 고객의 불편 사항을 경청합니다.
4
맞춤 솔루션 제시
파악한 니즈를 바탕으로 요금제 또는 결합상품을 제안합니다.
💡 "지금 요금보다 월 OO원 절감 가능합니다" — 숫자로 임팩트
5
가입 진행 & 개통 안내
고객 동의 후 필요 서류 안내 및 가입 절차를 설명합니다.
⚠️
과도한 혜택 과장은 민원으로 이어질 수 있습니다. 정확한 조건과 혜택을 안내하세요.

🔄 번호이동 절차 안내

1
기존 통신사 약정 잔여 기간 확인
위약금 발생 여부를 사전에 안내합니다. 고객 핸드폰에서 114 통화 시 해당 통신사 고객센터로 자동 연결됩니다.
2
번호이동 사전동의 처리
번호이동 사전동의 요청 완료 후, 전 통신사 안내 문자 확인 후 동의합니다.
문자에 "번호이동 사전동의" 내용이 없거나 문자가 없는 경우, 아래 사전동의 ARS 번호로 고객이 직접 전화하여 동의 처리합니다.
💡 전 통신사 문자에는 위약금·기존 혜택 내용만 있을 수 있음 — 그 경우 ARS로 직접 처리

②고객이 현재 이용 통신사를 잘못 알고 있는 경우 → 고객 핸드폰에서 114 통화로 확인 후 재요청
③전산 연동 오처리된 경우 → 다시 사전동의 요청 (실시간 처리이나 간혹 연동 누락·지연 발생)
3
개통 신청
고객/청구 정보, 요금제 선택, 인증 완료, 유심 번호 입력 후 개통 신청
💡 개통 오류 발생 시 — 안내 메시지에서 오류 항목을 확인하고 재입력
4
개통 확인 & 유심 삽입
개통 완료 메시지 확인 후, 단말기 유심 트레이(유심핀으로 추출)에 새 유심을 트레이 모양에 맞게 올려 삽입합니다.
일반적으로 즉시 인식하지 못하므로 2~3번 전원을 껐다 켜기 반복
🔍 유심 인식 불가 시 점검 항목
①개통 이력이 없는 통신사향 전용 새 단말기 → 사용 불가
②구형(eSIM 미지원)에서 신형(eSIM 지원)으로 변경 시 → KAIT 정보제공 동의 필요 (1588-3002 요청)
③유심 또는 단말기 유심 트레이 불량
⚠️
번호이동 개통 시 즉시 기존 통신사 서비스가 종료됩니다. 유심을 바로 교체해 주셔야 사용 가능합니다.

✅ 신규 개통 절차

1
요금제 선택
고객 니즈에 맞는 요금제 확정. 영업 상황에 맞게 고객용·대리점용 구분 중 주요 요금제 리스트에서 선택.
2
본인인증 & 가입 신청
신분증 확인 후 개통 시스템에 가입 정보 입력. 간편인증(네이버·카카오·PASS·신용카드), 안면인증 및 동의 처리.
3
개통 확인 & 유심 삽입
개통 완료 메시지 확인 후, 단말기 유심 트레이(유심핀 사용)에 새 유심을 모양에 맞게 올려 삽입합니다.
일반적으로 즉시 인식하지 못하므로 2~3번 전원을 껐다 켜기 반복
🔍 유심 인식 불가 시 점검 항목
①개통 이력이 없는 통신사향 전용 새 단말기 → 사용 불가
②구형(eSIM 미지원)에서 신형(eSIM 지원)으로 변경 시 → KAIT 정보제공 동의 필요 (1588-3002 요청)
③유심 또는 단말기 유심 트레이 불량
4
개통 완료 확인
통화·문자·데이터 연결 테스트 완료. 이용 안내 SMS 수신 확인.

🔧 유심 장착 방법 & 주의사항

1
단말기 전원 끄기
반드시 전원을 완전히 종료한 후 유심 교체를 진행합니다.
⚡ 전원 ON 상태에서 유심 교체 시 단말기 오류 발생 가능
2
유심 트레이 꺼내기
유심 핀(이젝터 핀)을 유심 트레이 홀에 꽂아 트레이를 꺼냅니다. 핀이 없을 경우 클립 사용 가능.
3
유심 방향 확인 후 장착
유심의 금색 칩이 아래를 향하도록, 모서리 잘린 방향 맞춰 트레이에 올려놓습니다.
💡 유심과 트레이의 모양(노치)을 맞추면 방향 실수를 줄일 수 있습니다
4
트레이 삽입 & 전원 ON
트레이를 부드럽게 밀어 넣고 전원을 켭니다. 화면 상단의 안테나 바가 표시되면 완료.
🚨
유심 핀 홀에 이물질이 들어가지 않도록 주의하세요. 무리한 힘으로 트레이를 삽입하면 단말기가 파손될 수 있습니다.

eSIM 지원 단말기 여부 확인

💡 핸드폰에서 개통 시 숫자를 길게 눌러 복사 붙여넣기 처리 / PC 개통 시 입력 주의
3
프로파일 다운로드
방법 1. 휴대폰 기본 카메라 실행 → QR 코드 스캔 → 프로파일 자동 다운로드
방법 2. 개통한 eSIM IMEI 기준으로 단말기에서 프로파일 설치 안내 메시지 수신 후 다운로드
💡 Wi-Fi 연결 필수 — 프로파일 다운로드에 데이터 연결이 필요합니다
4
개통 완료 확인
화면 상단의 안테나 바가 표시되면 완료.
🚨
단말기에서 eSIM과 유심 모두 사용하는 경우 동일 명의만 가능합니다. 다른 명의의 eSIM과 유심이 함께 있는 경우 자동 정지처리 됩니다.

✅ 답변: 네, 현재 사용 중인 단말기에 유심만 교체하면 됩니다. 단, 기존 통신사에 단말기가 잠금(락)되어 있는 경우 해제가 필요합니다. 개통 전 확인해 드립니다.
Q
"지금 위약금 있으면 어떻게 되나요?"
✅ 답변: 위약금은 기존 통신사에 고객님이 직접 납부하시는 부분입니다. 저희 요금제 절감액과 비교해서 결정하시면 됩니다. 고객센터(114)로 잔여 위약금 확인하시는 걸 도와드릴게요.
Q
"데이터 다 쓰면 어떻게 되나요?"
✅ 답변: 기본 데이터 소진 후에는 속도가 줄어들거나(QoS) 데이터 차단이 됩니다. 요금제별로 다르며, 추가 데이터 구매도 가능합니다. 선택하신 요금제 기준으로 설명해 드릴게요.
Q
"해외에서도 쓸 수 있나요?"
✅ 답변: 로밍 서비스를 신청하시면 해외에서도 사용 가능합니다. 요금제에 따라 기본 로밍 포함 여부가 다르니 확인해 드릴게요.
